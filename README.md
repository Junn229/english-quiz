# 中石油职称英语题库

中石油职称英语考试历年真题在线练习平台。

## 题型

- **词汇与结构** - 单选题（20题/卷）
- **语法** - 单选题（20题/卷）
- **阅读理解** - 5篇短文，每篇4题（20题/卷）
- **完形填空** - 1篇短文，10空（10空/卷）
- **翻译** - 英译中（A级/B级）

## 数据来源

- 2023年中石油职称英语考试真题（OCR识别）

## 使用方式

- 键盘快捷键：`A/B/C/D` 选择答案，`← →` 导航，`Enter` 确认
- 支持移动端响应式布局
- 点击左侧切换试卷和题型

## 部署

本项目部署于 GitHub Pages。
